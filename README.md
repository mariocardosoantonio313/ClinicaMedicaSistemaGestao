## Tecnologias
- Java JDK 23
- Java Swing (interface grafica)
- Arquitectura MVC
- Padrao DAO
- MySQL 8.0
- JDBC (mysql-connector-j-9.7.0)

## Como executar
1. Instalar JDK 17 ou superior
2. Instalar MySQL 8.0
3. Executar o script: clinica_medica_script.sql
4. Configurar src/database.properties
5. Abrir o projecto no NetBeans
6. Executar ClinicaMedicaoffc.java
